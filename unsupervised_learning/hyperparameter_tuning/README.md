# Hyperparameter Tuning

This directory contains exercises for hyperparameter tuning with Gaussian
processes and Bayesian optimization.

## Files

| File | Description |
|------|-------------|
| `0-gp.py` | Initialize a noiseless 1D Gaussian process |
| `1-gp.py` | Predict the mean and variance for sample points |
| `2-gp.py` | Update a Gaussian process with a new sample |
| `3-bayes_opt.py` | Initialize Bayesian optimization |
| `4-bayes_opt.py` | Calculate expected improvement for acquisition |
| `5-bayes_opt.py` | Optimize a black-box function |

## Requirements

- Python 3.x
- NumPy
- SciPy
