
# Probability

## Overview
This directory contains resources and implementations related to probability theory and statistical concepts in machine learning.

## Contents
- **Distributions**: Probability distributions (normal, binomial, Poisson, etc.)
- **Bayes' Theorem**: Conditional probability and Bayesian inference
- **Random Variables**: Discrete and continuous random variables
- **Expectation & Variance**: Statistical measures
- **Sampling Methods**: Monte Carlo, rejection sampling, importance sampling

## Topics Covered
- Probability fundamentals
- Joint and conditional probabilities
- Common probability distributions
- Central Limit Theorem
- Law of Large Numbers

## Usage
Each subdirectory contains practical implementations and examples in Python.

## Prerequisites
- Python 3.x
- NumPy
- Matplotlib (for visualizations)
