# Convolutions and Pooling

This module implements convolution and pooling operations on images from scratch using NumPy.

## Files

| File | Description |
|------|-------------|
| `0-convolve_grayscale_valid.py` | Valid convolution on grayscale images |
| `1-convolve_grayscale_same.py` | Same convolution on grayscale images |
| `2-convolve_grayscale_padding.py` | Convolution on grayscale images with custom padding |
| `3-convolve_grayscale.py` | Strided convolution on grayscale images |
| `4-convolve_channels.py` | Convolution on images with channels |
| `5-convolve.py` | Convolution on images using multiple kernels |
| `6-pool.py` | Max and average pooling on images |

## Requirements

- Python 3.x
- NumPy
