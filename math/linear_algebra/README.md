# Linear Algebra

This directory contains Python implementations and exercises for linear algebra concepts used in machine learning.

## Files

- **0-slice_me_up.py**: Array slicing operations
- **1-trim_me_down.py**: Array trimming operations
- **2-size_me_please.py**: Matrix shape calculation
- **3-flip_me_over.py**: Matrix transposition
- **4-line_up.py**: Vector addition
- **5-across_the_planes.py**: Matrix addition
- **6-howdy_partner.py**: Vector concatenation
- **7-gettin_cozy.py**: Matrix concatenation
- **8-ridin_bareback.py**: Matrix multiplication
- **9-let_the_butcher_slice_it.py**: Advanced array slicing
- **10-ill_use_my_scale.py**: Matrix shape function
- **11-the_western_exchange.py**: Matrix transpose function
- **12-bracin_the_elements.py**: Element-wise operations
- **13-cats_got_your_tongue.py**: NumPy concatenation
- **14-saddle_up.py**: NumPy matrix multiplication
- **100-slice_like_a_ninja.py**: Advanced slicing
- **101-the_whole_barn.py**: Matrix addition function
- **102-squashed_like_sardines.py**: Matrix concatenation function

## Requirements

- Python 3.x
- NumPy (for exercises 13+)

## Usage

Run individual scripts:

```bash
python3 0-slice_me_up.py
```

Or run with their corresponding main files:

```bash
python3 2-main.py
```
