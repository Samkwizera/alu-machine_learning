# ALU Machine Learning

This repository contains machine learning projects and exercises.

## Structure

- **math/**: Mathematical foundations for machine learning
  - **plotting/**: Data visualization exercises using matplotlib
  - **linear_algebra/**: Linear algebra implementations and exercises

## Requirements

- Python 3.x
- NumPy
- Matplotlib

## Usage

Navigate to the specific subdirectory to run the scripts:

```bash
cd math/plotting
python3 0-line.py
```
