# Pandas

This directory contains pandas exercises for working with tabular cryptocurrency exchange data.

## Files

- `0-from_numpy.py`: Creates a DataFrame from a NumPy array.
- `1-from_dictionary.py`: Creates a DataFrame from a dictionary.
- `2-from_file.py`: Loads a DataFrame from a delimited file.
- `3-rename.py`: Renames and converts timestamp data.
- `4-array.py`: Converts selected DataFrame columns to a NumPy array.
- `5-slice.py`: Slices selected columns and rows.
- `6-flip_switch.py`: Reverses row order and transposes the DataFrame.
- `7-high.py`: Sorts data by high price.
- `8-prune.py`: Removes rows with missing close values.
- `9-fill.py`: Fills missing values.
- `10-index.py`: Sets the timestamp column as the index.
- `11-concat.py`: Concatenates Bitstamp and Coinbase data.
- `12-hierarchy.py`: Builds a timestamp-first hierarchical index.
- `13-analyze.py`: Calculates descriptive statistics.
- `14-visualize.py`: Cleans, resamples, and plots daily data.

## Usage

Run each script from this directory with the required CSV files present:

```bash
python3 0-from_numpy.py
```
