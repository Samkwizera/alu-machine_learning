# APIs

This directory contains exercises for consuming public APIs with Python.

## Files

- `0-passengers.py`: Finds Star Wars ships that can hold a given number of passengers.
- `1-sentience.py`: Lists the home planets of all sentient Star Wars species.
- `2-user_location.py`: Prints the location of a GitHub user from the GitHub API.
- `3-upcoming.py`: Displays the next upcoming SpaceX launch.
- `4-rocket_frequency.py`: Counts SpaceX launches by rocket.

## Requirements

- Python 3.x
- requests

## Usage

Run each script from this directory:

```bash
python3 2-user_location.py https://api.github.com/users/holbertonschool
```
