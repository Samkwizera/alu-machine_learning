# Regularization

This project covers regularization techniques for supervised learning models.

## Files

| File | Description |
|------|-------------|
| `0-l2_reg_cost.py` | Calculate neural network cost with L2 regularization |
| `1-l2_reg_gradient_descent.py` | Update parameters using gradient descent with L2 regularization |
| `2-l2_reg_cost.py` | Calculate TensorFlow cost with L2 regularization losses |
| `3-l2_reg_create_layer.py` | Create a TensorFlow layer with L2 regularization |
| `4-dropout_forward_prop.py` | Perform forward propagation using dropout regularization |
| `5-dropout_gradient_descent.py` | Update parameters using gradient descent with dropout |
| `6-dropout_create_layer.py` | Create a TensorFlow layer using dropout |
| `7-early_stopping.py` | Determine when to stop training early |

## Requirements

- Python 3.x
- NumPy
- TensorFlow 1.x
