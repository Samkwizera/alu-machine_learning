# Classification

This project implements classification models from scratch using NumPy.

## Files

| File | Description |
|------|-------------|
| `0-neuron.py` | Single neuron for binary classification |
| `1-neuron.py` | Single neuron with private attributes |
| `2-neuron.py` | Neuron forward propagation |
| `3-neuron.py` | Neuron logistic regression cost |
| `4-neuron.py` | Neuron evaluation |
| `5-neuron.py` | Neuron gradient descent |
| `6-neuron.py` | Neuron training |
| `7-neuron.py` | Neuron training with verbose and graph options |
| `8-neural_network.py` | Neural network with one hidden layer |
| `9-neural_network.py` | Neural network with private attributes |
| `10-neural_network.py` | Neural network forward propagation |
| `11-neural_network.py` | Neural network cost |
| `12-neural_network.py` | Neural network evaluation |
| `13-neural_network.py` | Neural network gradient descent |
| `14-neural_network.py` | Neural network training |
| `15-neural_network.py` | Neural network training with verbose and graph options |
| `16-deep_neural_network.py` | Deep neural network initialization |
| `17-deep_neural_network.py` | Deep neural network with private attributes |
| `18-deep_neural_network.py` | Deep neural network forward propagation |
| `19-deep_neural_network.py` | Deep neural network cost |
| `20-deep_neural_network.py` | Deep neural network evaluation |
| `21-deep_neural_network.py` | Deep neural network gradient descent |
| `22-deep_neural_network.py` | Deep neural network training |
| `23-deep_neural_network.py` | Deep neural network training with verbose and graph options |
| `24-one_hot_encode.py` | One-hot encoding for label vectors |
| `25-one_hot_decode.py` | Decode one-hot matrices into labels |
| `26-deep_neural_network.py` | Deep neural network persistence with pickle |
| `27-deep_neural_network.py` | Deep neural network for multiclass classification |
| `28-deep_neural_network.py` | Deep neural network with selectable hidden activations |

## Requirements

- Python 3.x
- NumPy
- Matplotlib
