# Optimization

This project covers optimization techniques for supervised learning models.

## Files

| File | Description |
|------|-------------|
| `0-norm_constants.py` | Calculate feature-wise mean and standard deviation for data standardization |
| `1-normalize.py` | Normalize a matrix using provided means and standard deviations |
| `2-shuffle_data.py` | Shuffle two data matrices with the same random permutation |
| `3-mini_batch.py` | Train a restored TensorFlow model using mini-batch gradient descent |
| `4-moving_average.py` | Calculate a bias-corrected weighted moving average |
| `5-momentum.py` | Update variables using gradient descent with momentum |
| `6-momentum.py` | Create a TensorFlow momentum optimization operation |
| `7-RMSProp.py` | Update variables using the RMSProp optimization algorithm |
| `8-RMSProp.py` | Create a TensorFlow RMSProp optimization operation |
| `9-Adam.py` | Update variables using the Adam optimization algorithm |
| `10-Adam.py` | Create a TensorFlow Adam optimization operation |
| `11-learning_rate_decay.py` | Update a learning rate using stepwise inverse time decay |
| `12-learning_rate_decay.py` | Create a TensorFlow inverse time learning rate decay operation |
| `13-batch_norm.py` | Normalize an unactivated output using batch normalization |
| `14-batch_norm.py` | Create a TensorFlow batch normalization layer |
| `15-model.py` | Build and train a batch-normalized model with Adam and learning rate decay |

## Requirements

- Python 3.x
- NumPy
