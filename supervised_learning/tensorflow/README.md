# TensorFlow

This directory contains introductory TensorFlow 1.x exercises for building,
training, saving, and evaluating a neural network classifier.

## Files

- `0-create_placeholders.py`: creates input and label placeholders.
- `1-create_layer.py`: creates a fully connected neural network layer.
- `2-forward_prop.py`: builds the forward propagation graph.
- `3-calculate_accuracy.py`: calculates prediction accuracy.
- `4-calculate_loss.py`: calculates softmax cross-entropy loss.
- `5-create_train_op.py`: creates a gradient descent training operation.
- `6-train.py`: builds, trains, and saves the neural network.
- `7-evaluate.py`: restores a saved model and evaluates predictions.

## Requirements

- Ubuntu 16.04 LTS
- Python 3.5
- TensorFlow 1.x
- NumPy

Each Python file is executable and follows the project style requirements.
