# Error Analysis

This project covers error analysis metrics for supervised learning models.

## Files

| File | Description |
|------|-------------|
| `0-create_confusion.py` | Create a confusion matrix from one-hot labels and predictions |
| `1-sensitivity.py` | Calculate sensitivity for each class in a confusion matrix |
| `2-precision.py` | Calculate precision for each class in a confusion matrix |
| `3-specificity.py` | Calculate specificity for each class in a confusion matrix |
| `4-f1_score.py` | Calculate F1 score for each class in a confusion matrix |
| `5-error_handling` | Answer error handling strategy questions for bias and variance |
| `6-compare_and_contrast` | Identify the most important issue from training and validation results |

## Requirements

- Python 3.x
- NumPy
